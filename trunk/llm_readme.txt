.....:::::Linux Launcher Maker para Linux Launcher:::::.....

========== Uso ==========
-Escribe "llm.exe"
-Despues el nombre del archivo a escribir el destino ("LaunchBT.bat")
-Despues el nombre de la distribucion Linux ("BackTrack")
-Despues el nombre del archivo .iso ("bt3b141207.iso")
-Despues la pagina web desde donde se puede descargar
("http://backtrack.unixheads.org/bt3b141207.iso")
   
Ejemplo (para BackTrack):
 llm.exe LaunchBT.bat BackTrack bt3b141207.iso http://backtrack.unixheads.org/bt3b141207.iso
